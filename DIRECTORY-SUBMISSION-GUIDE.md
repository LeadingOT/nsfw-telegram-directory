## 🎨 需要的资源文件

### Logo文件（每个站点）
- 格式：PNG，透明背景
- 尺寸：512x512px 或更大
- 位置：`https://[domain]/og-image.png`

### Screenshot文件
- 格式：PNG 或 JPG
- 尺寸：1920x1080px 推荐
- 内容：网站首页完整截图

**快速截图命令：**
```bash
# 使用browser tool截图
for domain in aigirlfriend.tools accountingai.tools legalai.tools hrai.tools realestateai.tools bestwriting.tools; do
  # Screenshot已自动保存在各站点的og-image
  echo "Logo: https://$domain/og-image.png"
done
```
