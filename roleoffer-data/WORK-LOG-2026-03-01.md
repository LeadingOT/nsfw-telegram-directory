## Technical Details

### Anchor-Based Generation Strategy

**Problem:** Manual entry of 20,000 data points = error-prone + time-consuming

**Solution:** 16 carefully chosen anchor points → extrapolate with multipliers

**Formula:**
```python
def extrapolate_compensation(stage, location, role, level, percentile):
    baseline = eng_sf_series_a_late[level][percentile]
    stage_mult = get_stage_multiplier(stage, level, percentile)
    location_mult = LOCATION_MULTIPLIERS[location]
    role_mult = ROLE_MULTIPLIERS[role]
    return baseline * stage_mult * location_mult * role_mult
```

**Advantages:**
- ✅ Only 16 files to collect (not 20,000)
- ✅ Changes propagate automatically
- ✅ Consistent multipliers across dimensions
- ✅ Easy to add new cities/roles
